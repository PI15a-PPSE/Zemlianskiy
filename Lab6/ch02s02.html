Запуск шара

Для запуска игры (начало движения шара) необходимо вызвать метод Ball.start() и назначить интервал для метода Ball.move() с помощью window.setInterval(function{})

Example 2.2. Запуск шара

pa = new PlayingArea();
pa.ball.setDef();
pa.ball.start();
pa.ball.interval = window.setInterval(function() {pa.ball.move()}, 5);
