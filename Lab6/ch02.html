Инициализация игровой площадки

Для инициализации игровой площадки необходимо создать объект класса PlayingArea. После чего необходимо заполнить игровую площадку кирпичами c помощью метода PlayingArea.refreshArea(), а также установить шар на платформу с помощью функции класса Ball, обект которого находится в классе PlayingArea, Ball.setDef().

Example 2.1. Инициализация игровой площадки

pa = new PlayingArea();
pa.refreshArea();
pa.ball.setDef();
