Перемещение платформы

Для перемещения платформы за указателем мыши необходимо привязать метод Platform.move() к движению мыши с помощью метода$(document).mousemove(function(event) {})

Example 2.3. Перемещение платформы

pa = new PlayingArea();
$(document).mousemove(function(event) {      
        event = event || window.event;
        pa.platform.move(event.pageX || event.x);
})
